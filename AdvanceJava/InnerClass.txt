âœ… 2. Why anonymous inner class creates a new .class file but lambda does NOT?
Anonymous Inner Class

When you write:

Runnable r = new Runnable() {
    public void run() { }
};


Java creates a new class like:

OuterClass$1.class


Because anonymous inner class = real class created at compile time.

Lambda Expression

When you write:

Runnable r = () -> { };


Java does NOT create a new class file.

Instead, JVM uses:

invokedynamic


Which means the lambda is stored as a function, not a class.

âœ” No extra .class file
âœ” Modern JVM feature
